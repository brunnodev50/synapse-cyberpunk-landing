## 📄 Sobre o Projeto

Este projeto é uma **Landing Page** temática para uma loja fictícia de games chamada **Synapse Gaming**. 

O objetivo foi explorar a estética **Cyberpunk** utilizando apenas HTML5 e CSS3, sem dependências de frameworks JavaScript ou bibliotecas externas. O foco principal foi o uso avançado de CSS para criar uma atmosfera imersiva com luzes de neon, efeitos de glitch e interfaces futuristas.

## ✨ Funcionalidades e Efeitos Visuais

* **Efeitos de Neon & Glow:** Uso intensivo de `text-shadow` e `box-shadow` para criar a iluminação característica do estilo.
* **Animações CSS (Keyframes):**
    * *Flicker:* Simulação de lâmpadas de neon instáveis.
    * *Glitch:* Efeito de falha digital nos textos e títulos.
* **UI Design Futurista:**
    * Uso de `clip-path` para criar botões e cards com cortes angulares.
    * Estilização de inputs simulando terminais de comando.
* **Tratamento de Imagens:** Filtros CSS (`grayscale`, `hue-rotate`, `contrast`) que reagem dinamicamente ao `hover`.
* **Overlay de Scanlines:** Textura de linhas de varredura (estilo TV antiga/monitor CRT) sobre toda a página.
* **Design Responsivo:** Adaptável para diferentes tamanhos de tela.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:**
    * Flexbox e CSS Grid para layout.
    * CSS Variables (Custom Properties) para gerenciamento de paleta de cores.
    * Media Queries para responsividade.
* **Google Fonts:** Famílias *Orbitron* e *Share Tech Mono*.
